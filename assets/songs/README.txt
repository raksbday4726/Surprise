HOW TO ADD YOUR SONGS
─────────────────────
1. Copy your .mp3 files into this folder (assets/songs/)
2. Open story.html and update the songs array near the top of the <script>
3. Match the filename exactly, e.g.:
     src: "assets/songs/perfect.mp3"

Recommended file names to match the defaults already set:
  until-i-found-you.mp3
  perfect.mp3
  until-i-found-you-em.mp3

If you rename files, update the src in story.html to match.
